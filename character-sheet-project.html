Description

Created a full-stack desktop application for creating and managing custom character sheets for tabletop role-playing games. The application features a Java Swing-based graphical user interface designed to be both visually appealing and efficient. The back-end implements robust logic for handling character attributes using core object-oriented programming principles, with JSON-based data storage and the Jackson library for serialization and deserialization.
